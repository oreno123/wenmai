# 纹脉 Prompt 清单 — 复制粘贴到即梦/Jimeng

## 使用说明
- 每个 prompt 直接复制到即梦即可
- 出图后保存到 D:\桌面\纹样照片\
- 文件命名用前面的 [id]
- 出完图后跑 batch_all.py 自动提取元素
