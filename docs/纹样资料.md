# 中国传统纹样资料库

整理自多个来源，按类别分类，标注寓意与适用对称模式。

## 一、几何纹（最适合程序化生成）

| 纹样 | 特征 | 寓意 | 推荐对称模式 |
|------|------|------|-------------|
| **回纹** | 横竖折绕如"回"字 | 富贵不断头 | 4次旋转、四方连续 |
| **万字纹(卍)** | 卍字四端延伸连锁 | 万福万寿、绵延不断 | 4次旋转、四方连续 |
| **云雷纹** | 圆弧(云)与方折(雷)交替 | 天地交融 | 4次旋转、6次旋转 |
| **方胜纹** | 两个菱形交叠 |同心双合、同心协力 | 左右镜像、上下镜像 |
| **蒲纹** | 三向平行线交叉成六角形 | 编织、安宁 | 6次旋转、四方连续 |
| **曲水纹** | 水波+落花 | 流水不腐、生生不息 | 四方连续 |
| **垂鳞纹** | 鱼鳞排列 | 鳞甲护体 | 四方连续 |
| **环带纹/波曲纹** | 连续波浪起伏 | 山川河流 | 左右镜像、四方连续 |

## 五、程序化生成优先级建议

**第一批（几何感强，易用代码画）：**
1. 回纹 → 直线折绕，最简单
2. 万字纹 → 卍字重复
3. 云雷纹 → 圆弧+方折交替
4. 方胜纹 → 菱形交叠
5. 卷草纹 → S曲线+叶片

**第二批（曲线为主）：**
6. 祥云纹 → 如意云朵
7. 缠枝纹 → 波线骨架+花朵
8. 宝相花 → 多层花瓣
9. 莲瓣纹 → 花瓣排列

**第三批（动物形，需素材图）：**
10. 凤鸟纹、蟠螭纹、鱼纹等
